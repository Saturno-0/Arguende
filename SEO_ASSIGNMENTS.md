# SEO Assignments

## Cambios ya preparados en código
- `index.html`: metadatos SEO, Open Graph, Twitter Cards y JSON-LD (`CafeOrCoffeeShop`).
- `public/robots.txt`: reglas base para rastreo.
- `public/sitemap.xml`: sitemap inicial con la URL principal.

## Tareas asignadas para ti
1. Verificar que el dominio canonical sea `https://arguende.mx/` en:
- `index.html`
- `public/robots.txt`
- `public/sitemap.xml`

2. Validar y ajustar datos de negocio en `index.html`:
- Teléfono.
- Dirección exacta.
- Horarios.

3. Publicar y registrar en Google Search Console:
- Verificar propiedad del dominio.
- Enviar `https://arguende.mx/sitemap.xml`.

4. Validar rich results:
- Probar JSON-LD con Rich Results Test de Google.

5. Medición:
- Conectar GA4 y Search Console para seguimiento de consultas y CTR.
